# 📸 Instagram Prompt Automation System

A Full Stack Web Application that automates sending AI prompts via Instagram DM when users comment "PROMPT" on a reel.

---

## 🌐 Live Demo

- **Frontend:** https://insta-prompt-automation.vercel.app
- **Backend API:** https://insta-prompt-backend.onrender.com

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- JavaScript (JSX)
- CSS-in-JS

### Backend
- Python
- FastAPI
- Uvicorn
- SQLAlchemy
- Pydantic
- SQLite

### Deployment
- Vercel (Frontend)
- Render (Backend)
- GitHub (Version Control)

---

## ✨ Features

- 📊 Admin Dashboard with Stats
- 🎬 Add / Edit / Delete Reels
- 📝 Assign AI Prompts to Reels
- 💬 Auto DM when user comments "PROMPT"
- 📋 Activity Logs
